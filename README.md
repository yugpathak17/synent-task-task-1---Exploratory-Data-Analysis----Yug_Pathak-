#  Synent Task 2 – Exploratory Data Analysis (EDA)

## Problem Statement

The objective of this project is to perform Exploratory Data Analysis (EDA) on a real-world dataset to understand its structure, identify patterns, detect anomalies, and extract meaningful insights using statistical analysis and visualizations.

---

##  Dataset Details

**Dataset:** Customer Personality Analysis Dataset (Kaggle)

**Description:**
The dataset contains customer demographic details, purchasing behavior, and campaign response information.

* **Rows:** ~2,200+
* **Columns:** 29

---

##  Approach

The following steps were performed:

* Loaded and inspected the dataset.
* Checked data types and summary statistics.
* Analyzed missing values and distributions.
* Visualized relationships using histograms, boxplots, count plots, and correlation heatmaps.
* Identified trends, patterns, and potential outliers.

---

##  Results

*  Explored dataset structure and key statistics.
*  Identified data distributions and correlations.
*  Detected outliers and missing values.
*  Generated visual insights to support further preprocessing and machine learning.

---

##  Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

##  Outcome

This project provides valuable insights into the dataset through statistical analysis and visualization, forming a strong foundation for data preprocessing and machine learning model development.
